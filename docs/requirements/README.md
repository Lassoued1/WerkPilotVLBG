# Requirement Index

`REQUIREMENTS.csv` is a machine-readable index of the 106 explicit identifiers
in the approved cahier des charges. It is an index, not a replacement for the
source text.

Agents use it to:

- verify that a task names real requirement IDs;
- find the source section;
- associate a requirement with a delivery sprint and primary proof;
- detect accidental identifier loss when documentation changes.

The v2.1 inventory remains 106. Add, remove, move, or reinterpret an identifier
only after the contractual DOCX is formally rebaselined; a local change record
alone is not authoritative.
